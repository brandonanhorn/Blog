Exploratory Analysis

So you come into work on a Monday morning and find that your boss has tasked you with finding some insights on a current data set. You open the excel file and find that there is countless rows and columns that are filled with data. What do you do next?

Well, seens you are a trendy analyst or a naturally curious person, you decide to do some exploratory analysis to see what the data is all about.

As this is a personal interest project for your boss, and not a presentation, you have decided to use something trendy and use Python 3 and its wonderful libraries.

In this short blog, I will teach you how to begin the exploratory analysis process by using some of Python's built-in libraries to get you started.

To start, you are going to import the Pandas library, Matplotlib library, and the Seaborn library.

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

These are just a few of the libraries that are available for use within the Python language and in our case, will be the main libraries used for exploratory analysis.

Next you will call your bosses saved excel file name that you would have saved as a csv file.

example_1 = pd.read_csv('~/Documents/projects/project_1/data/act_2018.csv')

Now that you have loaded your file into the notebook, or text editor, you can begin the initial analysis with some Pandas functions!

Run the .head() function to show the first 5 rows, and all the column names to see what each column is called.

example_1.head()

Run the .tail() function to show the bottom of your DataFrame

example_1.tail()

Now that you have a basic understanding of your data, you should check to see if there is any null(0) values in your columns and what type of data is in your columns

example_1.info()

In this data set you can see that there is no null values, and that there is a mixture of object type data, and float type data and int type data.

Next you can use the .describe().T function to see things like min and max values in each column, as well as, std. deviation and the mean in each column

example_1.describe().T

You can see with just a few simple functions from the Pandas library, we have a gained a significant understanding of our numerical data.

And now its time to bring in the next library, Matplotlib. It is a library used for plotting your numerical data.

Lets get started first by plotting each column in simple histograms, maybe it will provide some insight on if our data is normalized or not.

plt.figure(figsize=(4,4))
plt.hist(example_1['English']);

plt.figure(figsize=(4,4))
plt.hist(example_1['Math']);

plt.figure(figsize=(4,4))
plt.hist(example_1['Reading']);

plt.figure(figsize=(4,4))
plt.hist(example_1['Science']);

plt.figure(figsize=(4,4))
plt.hist(example_1['Composite']);

You can use Matplotlib to quickly plot a few different styles of plot, but for exploratory, I like to keep things simple with either histograms or scatterplots because they visualize the data effeciently.

Remember, when doing exploratory analysis, you don't need to make things pretty, you just need to make things easy for you to read and understand.

The last visual image I like to use is a heatmap, it shows correlation among our data cleanly and effectively. This is where Seaborn library is most effective.

fig, ax = plt.subplots(figsize=(6,6))  
sns.heatmap(example_1.corr(), annot=True, cmap="icefire");

You can see the value of the heatmap rightaway because of how the human eye interprets color and correlation. Using the icefire colorway you are able to see that Participation is negatively correlated with all of the scores provided by the ACT.

And thats it. In probably less then 20 minutes, expert or not in Python, you are able to quickly explore your data and see what trends and interpretations you can make from your data. You can report back to your boss what you are seeing, or if you are feeling like you'd want to go farther, check back soon for the next blog!
